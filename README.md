# MedXpert 🏥

MedXpert is a mobile-based healthcare assistance application built with Flutter. It provides quick and reliable access to essential medical services by integrating online doctor consultations, emergency blood requests, pharmacy ordering, and ambulance services into a single, user-friendly platform.

---

## 🚀 Features

- 👨‍⚕️ Online Doctor Consultation
- 🩸 Emergency Blood Requests
- 💊 Pharmacy Ordering
- 🚑 Ambulance Assistance
- 👤 User Profile Management
- 🔐 Secure Authentication
- 📱 Modern and Intuitive UI

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend & Authentication:** Firebase
- **Platform:** Android & iOS (Cross-platform)
